藥劑部班表系統 V1.6.11

這是一個專為藥劑部設計的自動化班表管理與換班系統。

🚀 系統功能

不用登入就可即時檢視全月班表。

帳號管理：提供人員管理與帳密重置等功能。

班別管理：班別代號、人日數與國定假日設定。

換班系統：三階段核定 申請人發起 -> 同仁初步核定 -> 組長最終核定。 自動判斷班別類型（如 A1A2, A3, P 班等），並支援「整段週期換班」或「單日換班」。

紀錄系統：可查詢與核定所有換班紀錄。

預假系統：支援每月 15 號自動抽籤分配假日及平日休假名額。 名額控管： 設定平日與假日的可休人數上限。  

排班系統： CSV 匯入/匯出： 支援透過 CSV 大量更新班表資料，具備匯入預覽與衝突比對模式，可點擊比對新舊班別。漏排提醒：根據設定的常態班規則，提醒尚未排定的必要班別。

管理報表：提供人日數統計、夜班費計算及 28 天（雙週）班別代碼報表。且能根據藥師分類（勞基法/非勞基法）自動生成預設班別並檢核一例一休。

🛠️ 技術架構

前端框架：React 18

編譯工具：Vite

樣式工具：Tailwind CSS

圖示庫：Lucide React

部署平台：GitHub Pages (經由 GitHub Actions 自動部署)
