藥劑部班表系統

這是一個專為藥劑部設計的班表管理、預假與換班系統。

🚀 系統功能

不用登入就可即時檢視全月班表
長時間未使用請重整頁面，避免被瀏覽器紀錄誤導。

● 帳號管理：提供人員管理與帳密重置等功能。
● 班別管理：班別代號、人日數與國定假日設定。
● 換班系統：
- 換班單申請：點選人員班別即可送出換班單，支援「單日換班」或「多人換班」(多人班別形成換班迴圈)。
- 支援「整段週期換班」，自動判斷班別類型（如 A1A2, A3, P+8天連休等），請注意周日A1#與A2#不計入整段換班，需另外申請。
- 三階段核定：申請人發起 -> 換班單中所有同仁同步核定(不用一關一關過，也可看到卡在哪一關) -> 組長最終核定。 
● 紀錄系統：可查詢與核定所有換班紀錄，以換班的班別日期為準(非申請換班單的時間)，搜尋區間查詢。
● 預假系統：點選日期即可預假，再次點選取消，每月 15 號自動抽籤休假名額。
● 排班系統：
- CSV上傳班表：發布後自動下載原版班表備份，若需回復上一步，將原版班表重新上傳即可
- 檢核功能：具備匯入預覽與衝突比對模式，可點擊比對新舊班別。並檢核班表內容與上傳月份，阻擋非該月份班表上傳。
- 漏排提醒：根據設定的常態班規則，提醒尚未排定的必要班別。
- 勞基法及時檢核：開啟後設定雙周區間起始日，能根據藥師分類（勞基法/非勞基法）自動檢核一例一休。
● 管理報表：提供人日數統計、夜班費計算及 28 天（雙週）班別代碼報表。
