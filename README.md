藥劑部班表系統 V1.6.11

這是一個專為藥劑部設計的自動化班表管理與換班系統。

🚀 系統功能

自動化排班：根據藥師屬性（勞基法/非勞基法）自動生成預設班別。

管理報表：提供人日數統計、夜班費計算及 28 天（雙週）班別代碼報表。

預假系統：支援每月 15 號自動抽籤分配假日及平日休假名額。

換班申請：提供單日換班及「P 班/A 班」整段換班功能，並具備組長審核流程。

🛠️ 技術架構

前端框架：React 18

編譯工具：Vite

樣式工具：Tailwind CSS

圖示庫：Lucide React

部署平台：GitHub Pages (經由 GitHub Actions 自動部署)

📦 如何安裝與開發

複製儲存庫：
git clone https://github.com/您的帳號/儲存庫名稱.git

安裝套件：
npm install

啟動開發伺服器：
npm run dev

編譯生產環境：
npm run build
